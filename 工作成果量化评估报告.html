★立项初衷 · 为什么要建这个项目

不是又一个通用 AI 工具，而是面向影视工作室、以「生产资料私有化 + 全链路可控」为根本诉求的定制创作中枢。下面从十个维度，逐条说明设计初衷与对应的具体实现（均已落地于代码）。
🏠01数据主权 · 数据不出域
设计初衷 · 影视与商业素材是工作室的核心资产，绝不能存在第三方平台、绝不能随生成外发——必须满足版权归属、商业保密与合规要求。
已落地实现 ·
自托管 MySQL + MinIO/S3产物永久硬锁自有存储 assertMinioOnlyWrite第三方产出(14天临时URL)自动转存自有多存储后端 MinIO/S3/Forge 可选图/音/视持久化开关可分别配置同源去重 + 不可达回退
🛡️02产出物防泄露 · 八重纵深防护
设计初衷 · 珍贵创作产出与商业数据，从存储、传输、访问到分享层层设防，让越权与盗链无门可入、敏感操作全程可追责。
已落地实现 ·
下载鉴权网关(登录态+所有权/项目角色校验)S3 预签名短时链接(防长期暴露)媒体代理隔离(隐藏真实存储地址防盗链)强制服务端中继限时分享 + 防暴力破解 + 仅所有者可公开所有者/编辑者/只读者 分级权限
💧03身份水印溯源 · 泄露可追责
设计初衷 · 即使内容被下载或截屏录屏，也要能溯源到具体的人、形成威慑——这是「人」的防线，补足技术防线之外的最后一环。
已落地实现 ·
下载身份水印(邮箱/ID+时间戳烧录进图/视频)页面级身份水印(全站淡叠加，防截图录屏)开发者工具阻断(防源码/接口/资源被扒)敏感操作(下载/删除/分享/密钥)全留痕
⚡04算力自有 · 开闭互补（核心壁垒）
设计初衷 · 不被云算力账号绑架，让算力与数据全程留在自有环境；私有 GPU 的无限定制 + 闭源旗舰的开箱即用，形成难以复制的能力壁垒。
已落地实现 ·
原生深度对接自建 ComfyUI / 私有 GPU 集群多服务器负载均衡 · 任务轮询打散显存智能释放 freeVramAfterRun本地×官方云 弹性互补 · 成本动态平衡接口监测与连接健康自动早退容错管理员多服务器并发压力测试
✨05模型统一聚合 · 开闭同台
设计初衷 · 把 6 大服务商、150+ 模型汇于一处，统一接口、统一密钥、统一计价——创作者无需关心后端差异，且杜绝参数错误、价格透明。
已落地实现 ·
kie.ai / Poyo / Manus Forge / Higgsfield / ComfyUI 聚合图像 54 · 视频 75 · 对话 26 + 音频多类逐字段对照官方文档接入防参数错误价格按官方计价透明标注管理后台按节点分组模型使能开关
🔑06密钥 · 额度 · 成本治理
设计初衷 · 多租户场景下，必须做到额度隔离、成本可控可对账、防越权白嫖——谁用谁的额度、花了多少，都要算得明白。
已落地实现 ·
三级密钥优先级：临时(用户)>分配(管理员)>公用(平台)密钥 AES-256-GCM 加密落库 · 永不明文公用 key 白名单门控防越权白嫖多租户额度隔离 · 互不挤占实时点数预估 + 预算管控面板(超限自动暂停)kie/Poyo 余额实时查询 + 消费趋势快照
👁️07完备权限 · 全链路审计
设计初衷 · 工作室需要清楚地控制「谁能用、用哪些、做了什么」，并让每一步敏感操作可回溯、可追责、可对账。
已落地实现 ·
OAuth + 邮箱密码(scrypt 加盐) 双认证管理员 L1~L4 四级权限分权协作 所有者/编辑者/只读者 三级28 类操作全链路审计日志预估点数随成败写入审计可对账操作/ComfyUI 日志一键导出 CSV
🖧08局域网 / 内网协作 · 端到端加密
设计初衷 · 工作室常处于内网、离线或物理隔离环境，团队需要在不依赖公网的局域网内也能实时共创与沟通；且沟通内容隐私，服务端不可读。这正是选择「自托管 + 内网优先」而非公有 SaaS 的根本原因。
已落地实现 ·
局域网/内网聊天与房间(离线环境可协同)实时多人协作 · 节点秒级同步 · 协作者光标可见私聊 DM + 群聊房间(项目级群组)端到端加密(公钥发布+客户端加密，服务端不可读)公网隧道 cloudflared 按需暴露 + 隧道级白名单门控Socket.IO 同时挂主/隧道回环服务器保实时不丢
🗄️09素材自主管理 · 永不丢失
设计初衷 · 团队创作资产要统一沉淀、跨项目复用、可追溯来源、误删可恢复——素材是反复使用的生产资料，不能散落、不能丢。
已落地实现 ·
跨项目个人/团队素材仓库生成资产自动入库(来源/模型/项目可追溯)批量上传自动识别图/音/视软删除可恢复 + 管理员硬删除素材回填(扫描画布把漏库产物补录，幂等)项目自动封面 + 仓库素材统计看板
🏢10私有定制 · 平滑运维
设计初衷 · 这不是通用 SaaS，而是为工作室量身定制、可按行业裁剪、品牌自主、可平滑升级的私有化创作底座。
已落地实现 ·
品牌 LOGO/图标/加载页/品牌位全可替换功能模块按业务裁剪节点化架构按行业(影视/电商/营销/教育)扩展管理后台一键系统更新 + 自动数据库迁移迁移经真实数据库幂等验证Windows / Docker / 裸机 一键部署

  • 🔁
    轮询 + WebSocket 实时进度

  • 📥
    流式下载防 OOM + 产物硬锁自有存储
    Content-Length 预检 + 流式累计超限即 cancel(图30MB/视频200MB)；产物一律重传 MinIO/S3，绝不返回 ComfyUI 原始 URL。

    GGUF 加载器动态转换
    检测 .gguf 自动把 CLIPLoader→GGUF、UNETLoader→GGUF 并剔除不兼容参数，用户无感。
  • 🩺
    错误三层回溯 + 中文化诊断
    从 execution_error/node_errors/traceback 提取真因；按维度数(3584=Qwen/4096=T5/2048=SDXL)推荐架构、缺节点推荐插件——给可操作中文。
  • 🔀
    UI→API 工作流自动转换 + 参数绑定自动上游填充
    拖 PNG/JSON 即用：解析 links、追踪 Reroute、虚拟节点剔除、一次性聚合错误；URL 自动上传为本地文件名。
  • 📊
    多地址轮询压测 + per-server 分桶 + Crystools GPU 监控
    worker 池 round-robin、每秒时间序列采样、AbortController 立即停止；WebSocket 被动订阅物理 GPU 利用率/显存/温度。
  • 🔒
    SSRF 防护 + 凭证保护
    拒绝云元数据端点(IMDS)、拒绝 URL 内用户名密码、API Key 走 Header；模板 payload 净化。
